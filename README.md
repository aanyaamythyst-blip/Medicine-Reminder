# 💊 Medicine Reminder App

A simple **desktop medicine reminder** built with Python.
This application helps users remember to take their medicine at a specific time by showing a pop-up notification.

It’s a beginner-friendly project created to practice **Python GUI development and time-based automation.**

---

## ✨ Features

* Enter the **medicine name**
* Select **dose type** (Morning or Evening)
* Choose a **reminder time (HH:MM)**
* **Start** or **Stop** the reminder anytime
* Pop-up alert when it's time to take your medicine

---


---

## 🛠️ Technologies Used

This project uses the following technologies:

* **Python** – Main programming language
* **Tkinter** – Used to create the graphical user interface (GUI)
* **Datetime module** – Used to track and compare the current time with the reminder time
* **MessageBox (Tkinter)** – Displays reminder pop-up alerts

---

## 🚀 How to Run the Project

1. Make sure **Python 3** is installed on your computer.

2. Clone this repository:

```bash
git clone https://github.com/yourusername/medicine-reminder-app.git
```

3. Go to the project folder:

```bash
cd medicine-reminder-app
```

4. Run the program:

```bash
python reminder.py
```
